Design Concept Drawings

Summary

Many projects in my current job starts with hand sketches, and they help me and my team convey ideas and specific mechanical design benefits. These drawings are used during customer communication, internal project pitching, brainstorming, and patent applications.

This illustration was done to explain the tool I design to assist in the assembly process of a custom flexible silicone sleeve cover. The tool is an expansion pliers fitted with 90 degrees offset fins that evenly stretch the thin silicone based cover over the connector that needed higher IP rating.

This sketch was part of the reference material for a patent filed under a Design for Automation review project. I designed the components according to DFA principals such as in-line assembly, in which parts can be put together in as few axial direction as possible, thereby reduce automation complexity.

This concept is part of the same project as the last sketch. I created this drawing to illustrate a new fastening method where all of the wires that goes into a terminal block can be terminated and locked in or released at the same time. It could drastically reduce installation time and reduce terminatioin error due to insufficient tighting with tradition individual screw-tight mechanism.

As part of a user experience brainstorming session, I came up with this drawing to illustrate features that can be added to better align and confirm the bulkhead and housing latching action.

This drawing was done for my colleague's project exploring a new type of current sensor using micro 3D printed conductive material onto wires that get coiled around cables that needs sensing.
